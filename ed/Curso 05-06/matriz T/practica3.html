Práctica sobre construcción y uso básico de TDAs.

Objetivo.

El objetivo de esta práctica es el de familiarizarse con la definición, especificación e implementación de un TDA.
Así mismo, tener un primer contacto con el uso de los TDAs de la STL más sencillos.

Tareas a realizar.

Se debe:
  1. revisar cuidadosamente el conjunto de operaciones de que dispone el tipo de dato,
  2. realizar una especificación adecuada para cada una de ellas,
  3. buscar una representación adecuada para el tipo de dato que permita que las operaciones se puedan realizar de la forma más eficiente desde el punto de vista de tiempo y/o espacio,
  4. definir una función de abstracción y los invariantes que se deben cumplir para esa representación.

Definición del problema.

Construir una clase matriz<T> donde T debe ser una clase con una relación de orden entre sus elementos.
Se debe proponer una representación válida e indicar la función de abstracción y los invariantes de la representación.
La clase debe tener las siguiente operaciones (para las que hay realizar una especificación)
  1. Constructor por defecto de la clase. Se le indica el número de filas y de columnas.
  2. Constructor basado en un dato de tipo T. Crea una matriz con un número de filas y número de columnas y en cada posición tiene el dato de tipo T
  3. Constructor de copia.
  4. Destructor
  5. ==, !=
  6. <<
  7. int nfilas() const. Devuelve el número de filas de la matriz
  8. int ncols() const. Devuelve el número de columnas de la matriz
  9. void nfilas(int n). Cambia el número de filas. Si n es menor que el número de filas actual, debe conservar las n primeras.
  10. void ncols(int n). Cambia el número de columnas. Si n es menor que el número de columnas actual, debe conservar las n primeras.
  11. T get(int nf, int nc) const. Devuelve el elemento que se encuentra en la posición (nf,nc)
  12. void set(int nf, int nc, const T &dato). Asigna un valor dato en la posición (nf,nc)
  13. T minimo() const. Devuelve el valor mínimo de la matriz
  14. T minimo_fila(int fila) const. Devuelve el mínimo para una fila i
  15. T maximo() const. Devuelve el valor máximo de la matriz
  16. T maximo_fila(int f) const. Devuelve el máximo de una fila i
  17. bool simetrica() const. Devuelve un valor lógico que indica si la matriz es o no simétrica.
  18. matriz<T> traspuesta() const. Devuelve la matriz traspuesta.
Se deben probar todas y cada una de las operaciones usando ejemplos.

Además, se deben resolver los siguientes problemas (con la cabecera que se indica):
  • ejemplo1: Construir una función que determine si una matriz de enteros de tamaño nxn es "mágica". Donde mágica significa que la suma de cada fila, de cada columna y de cada diagonal es la misma.
    • bool magica((const matriz<int> &entrada)
  • ejemplo2: Construir una función que obtenga los mínimos de cada fila de una matriz y los devuelva en una lista de forma ordenada.
    • list<T> minimos_fila(const matriz<T> &entrada)
  • ejemplo3: Construir una función a la que se le pase una lista de matrices de enteros y que las multiplique.
    • matriz<int> multiplicar(const list<matriz<int> > &entrada)
  • ejemplo4: Construir una función a la que se le pase una matriz y que devuelva una lista ordenada de tripletas {dato, fila, columna}. Para ello se debe utilizar una cola con prioridad.
    • list<coordenadas> ordenada(const matriz<T> &entrada)

Entrega

Se debe entregar, usando GAP, la siguiente información:
  • matriz.h: fichero de cabecera de la clase que debe contener la definición de la clase y la especificación de cada una de las operaciones.
  • matriz.template: fichero que debe contener la implementación de cada una de las operaciones.
  • ejemplos.h: Fichero de cabecera para los ejemplos.
  • ejemplos.cpp: Implementación de las funciones ejemplo.
  • ejemplo1.cpp: Programa de prueba del ejemplo1
  • ejemplo2.cpp: Programa de prueba del ejemplo2
