Diseño y evaluación de computadores

Práctica 1: Uso de recursos relacionados con la asignatura

  • Benchmarks:

    1. http://www.phystech.com/download/ubench.html: Potente aunque ya desfasado benchmark para sistemas operativos tipo Unix. Incluye un ranking.

    2. http://lbs.sourceforge.net: Colección de benchmarks sintéticos y no-sintéticos para Linux.

    3. http://www.futuremark.com/benchmarks/3dmark06/introduction/: Site del popular benchmark para Windows, PC-Mark. Incluye un ranking.

  • Software:

    1. http://www.tucows.com: pWeb de software clasificado y actual para múltiples plataformas (Windows, MAC, Linux, etc)

    2. http://www.cdlibre.org/: Completo catálogo de software libre para Windows.

  • Hardware:

    1. http://foro.hardlimit.com/index.php: Foros sobre hardware orientados principalmente a overcloking y modding.

    2. http://www.noticias3d.com: Site sobre hardware orientado principalmente a tarjetas gráficas, contiene artículos, noticias y foros entre otros.

  • Publicaciones:

    1. http://www.telepolis.com/nwl/digital/tecnologia.htm: Boletín diario sobre hardware, software, electrónica comercial, etc.

    2. http://www.custompc.co.uk: Web de la revista Custom PC publicada en Reino Unido, orientada a rendimiento y personalizació (modding) de equipos.

    3. http://www.revistabegins.org/: Revista sobre Software Libre y Código abierto.
